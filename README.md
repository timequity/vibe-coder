# Vibe Coder

Describe your idea, get a deployed product. All Craft Coder skills + brainstorming, validation, and deployment automation.

## Features

**70+ skills for vibe coding:**

- **All Craft Coder skills** included
- **Brainstorming** — Refine your idea
- **Idea Validation** — Validate before building
- **Stack Selection** — Choose the right tech
- **UI Generation** — Auto-generate interfaces
- **Feature Building** — Build features fast
- **Auto Testing** — Automated test generation
- **Security Checks** — Built-in security
- **Deploy Automation** — One-click deploy

## Commands

| Command | Description |
|---------|-------------|
| `/vibe-coder:ship` | From idea to deployed app |
| `/vibe-coder:idea` | Start with a simple idea |
| `/vibe-coder:brainstorm` | Refine your concept |
| `/vibe-coder:build` | Build from validated idea |
| `/vibe-coder:add` | Add a feature |
| `/vibe-coder:preview` | Preview your app |
| `/vibe-coder:deploy` | Deploy to production |

## Installation

### Option 1: Download ZIP (Recommended)

1. Go to [Releases](../../releases)
2. Download the latest `vibe-coder.zip`
3. Extract to `~/.claude/plugins/vibe-coder/`

```bash
mkdir -p ~/.claude/plugins
cd ~/.claude/plugins
unzip ~/Downloads/vibe-coder.zip -d vibe-coder
```

### Option 2: Git Clone

```bash
cd ~/.claude/plugins
git clone https://github.com/timequity/vibe-coder
```

### Verify Installation

Restart Claude Code and type `/vibe-coder:ship`
