# Idea to MVP Skills

All skills work automatically. You don't need to invoke them directly.

## Ideation Phase

### brainstorming
Refine ideas through Socratic dialogue. One question at a time.

### idea-validation
Validate problem/solution fit. Creates internal PRD.

## Planning Phase

### stack-selector
Automatically choose tech stack based on requirements.

### db-designer
Design database schema from requirements.

## Building Phase

### ui-generator
Create UI components from descriptions.

### api-generator
Generate API endpoints from requirements.

### feature-builder
Add features incrementally with proper architecture.

## Quality Phase

### test-driven-development
TDD cycle: write tests first, then implement.

### auto-testing
Run full test suite automatically.

### security-check
OWASP validation, security scanning.

### code-review-auto
Automatic code review, fix issues before showing.

## Polish Phase

### frontend-design
Distinctive UI avoiding "AI slop" aesthetics.

### theme-factory
Consistent colors, typography, design system.

### verification-gate
Final validation before completion.

## Deployment Phase

### deploy-automation
Choose platform, configure hosting, deploy.

### debugging-auto
Automatic issue detection and fixing.

## The Magic

You see: Simple questions and "Done!"

Under the hood: 15+ skills working together in a validated pipeline.

Every change runs through: TDD → Tests → Security → Review → Verify
